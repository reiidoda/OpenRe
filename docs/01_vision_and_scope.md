# Vision and Scope

## Vision
Build an open reference workbench for evaluating and safely operating multimodal agents.

## In scope (v1)
- CLI-first benchmark runner.
- Config-driven agent definitions.
- Structured traces and exportable reports.
- Approval gates for risky actions.

## Out of scope (v1)
- Consumer chat UI.
- Autonomous destructive actions.
- Full multi-cloud deployment tooling.
