# [M0] Add local JSON trace sink

## Summary
Persist structured trace events per run in local JSONL artifacts.

## Acceptance Criteria
- Trace sink writes newline-delimited JSON.
- `run_id`-scoped trace file is created for each run.
- Event schema includes event id, kind, timestamps, and payload.
